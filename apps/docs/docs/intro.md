# Welcome to your Yese Creations handover guide

This is your private reference for running the shop day-to-day. It only
shows up here, behind your admin login — nobody browsing yesecreations.com
can see it, and it isn't indexed by search engines.

There are three guides:

- **[Managing products](/managing-products)** — adding new pieces, changing
  prices, marking something sold out or as a bestseller, uploading photos.
- **[Homepage & content sections](/homepage-and-content)** — editing the
  text, images and sections that make up the homepage itself.
- **[Orders & Stripe basics](/orders-and-stripe)** — how an order gets from
  "customer paid" to "I know what to post out," and where to look if a
  payment seems to have gone wrong.

## The two places you'll work

Almost everything is done from one place: **the admin panel**, at
`/admin` — the same login you used to get to this guide. It's where you add
products, edit homepage text, and look up orders.

The shop itself (the part your customers see) is a separate website. You
won't normally need to touch it directly — everything on it (products,
prices, photos, homepage text) is pulled live from what you enter in the
admin panel.

## If something feels stuck

Nothing you do in the admin panel goes live to customers until you click
**Save** (or **Publish**, on pieces of content that have that button
instead). If a change doesn't seem to show up on the live site, the most
common reason is simply that it wasn't saved yet — check the top of the
page you were editing for an unsaved-changes indicator.

If you ever get properly stuck, the person who built this site is the best
first call — this guide covers the day-to-day, not troubleshooting a broken
build or server.
